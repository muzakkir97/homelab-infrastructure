### Gilgamesh Feature Roadmap

```
CURRENT STATE
└── Memory, Smart Routing (Haiku/Sonnet), Web Search, /update command

IN PROGRESS
└── Inline Keyboard Menu (Status, Metrics, Temps, Storage submenus)

PLANNED COMMANDS
├── /temps      → CPU, GPU, NVMe temperatures
├── /storage    → Disk usage across pools
├── /alerts     → Recent Alertmanager alerts
├── /logs       → Fetch logs from Loki
├── /backup     → Backup status check
├── /speedtest  → Network speed test
├── /wake       → Wake-on-LAN for devices
├── /cost       → Claude API costs this month
├── /memory     → What Gilgamesh remembers
├── /clear      → Clear conversation memory
└── /help       → List all commands

SMART FEATURES
├── Proactive Alerts    → Gilgamesh messages YOU when issues arise
├── Daily Summary       → Morning health check message
├── Anomaly Detection   → Unusual traffic/behavior alerts
└── Image Analysis      → Send error screenshots for explanation

SECOND BRAIN (Phase 7H)
├── Obsidian vault synced via Nextcloud
├── claude-obsidian for automated wiki building
├── Session summaries written to Obsidian
├── /save command → Extract key ideas to wiki
├── /autoresearch → Deep research with citations
└── Replaces RAG phase — claude-obsidian IS the RAG

AI ENHANCEMENTS (After Phase 11)
├── Local LLM Fallback  → Use Ollama if Claude API fails
└── Hybrid Routing      → Simple queries → Ollama, complex → Claude
```
